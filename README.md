## Required Dependencies
```
pandas
matplotlib
seaborn
streamlit
babel
```

## create virtual env
```
python -m venv dico
```
## activate
```
myenv\Scripts\activate
```

## Setup Environment - Shell/Terminal
```
mkdir dashboard
mkdir data
cd dashboard
cd notebook
pip install streamlit
pip install -r requirements.txt
pip freeze requirements.txt

```

## Run steamlit app
```
streamlit run dashboard.py
```
